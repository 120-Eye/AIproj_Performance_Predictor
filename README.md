# Student Performance Visualization Dashboard

## 🚀 Quick Start

**Local Testing:**
```
# Open in browser (no server needed for basic functionality)
start index.html
```

**Full Deployment (with AI insights):**
1. Push to GitHub:
```
cd C:/Users/hirak/Desktop/student-dashboard
git init
git add .
git commit -m "Initial student dashboard"
git remote add origin https://github.com/YOUR_USERNAME/student-dashboard.git
git push -u origin main
```

2. Deploy to Vercel:
- Go to [vercel.com](https://vercel.com)
- Import GitHub repo
- Add Environment Variable: `GEMINI_API_KEY`
- Deploy!

## 🌟 Features
- ✅ Professional CRM-style UI (responsive)
- ✅ Dynamic subjects/marks input
- ✅ localStorage persistence
- ✅ Chart.js visualizations (bar/line/pie)
- ✅ Real-time basic analytics
- ✅ Google Gemini AI insights
